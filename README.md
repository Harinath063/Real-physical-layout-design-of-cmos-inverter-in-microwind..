 # CMOS INVERTER PHYSICAL LAYOUT USING MICROWIND 3.5

## 📌 PROJECT TITLE

Design and Implementation of CMOS Inverter Physical Layout using Microwind Tool

## 📌 TOOL USED

Microwind 3.5

## 📌 TECHNOLOGY

CMOS 1.2µm Technology (5V)

## 📌 DESCRIPTION

This project presents the physical layout design of a CMOS inverter using Microwind 3.5. The layout consists of both PMOS and NMOS transistors arranged in a standard CMOS configuration.

The PMOS transistor is placed inside the N-well region and connected to Vdd, while the NMOS transistor is placed outside the well and connected to Vss (ground). The gates of both transistors are connected together using polysilicon to form a common input.

The drains of both PMOS and NMOS are connected using metal to form the output node. The sources are connected separately to Vdd and Vss respectively.

## 📌 WORKING PRINCIPLE

* When input is LOW (0), PMOS turns ON and NMOS turns OFF → Output is HIGH (1)
* When input is HIGH (1), NMOS turns ON and PMOS turns OFF → Output is LOW (0)

## 📌 LAYOUT FEATURES

* Common polysilicon gate for both transistors (input)
* Shared drain region connected via metal (output)
* Proper Vdd and Vss connections
* N-well used for PMOS transistor
* Contacts used for metal-to-diffusion connections

## 📌 FILE FORMAT

The layout is saved in .MSK format (Microwind layout file)

## 📌 APPLICATIONS

* Basic digital logic circuits
* Building block for NAND, NOR, Flip-Flops
* Used in VLSI and ASIC design
